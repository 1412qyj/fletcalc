# 计算器 App 需求评审 + 研发方案书

---

## 一、项目概述

- **项目名称**: FletCalc (Python 计算器)
- **项目类型**: Android 移动应用
- **技术栈**: Python + Flet + Flutter
- **目标用户**: 日常计算需求用户

---

## 二、需求分析

### 2.1 功能需求

| 序号 | 功能模块 | 描述 | 优先级 |
|------|----------|------|--------|
| 1 | 基本运算 | 加、减、乘、除 | P0 |
| 2 | 清除功能 | C (清空)、CE (清当前) | P0 |
| 3 | 小数点支持 | 支持小数运算 | P0 |
| 4 | 百分比计算 | % 百分比 | P1 |
| 5 | 正负切换 | ± 反转符号 | P1 |
| 6 | 历史记录 | 显示最近 10 条计算历史 | P2 |

### 2.2 UI 需求

- **布局**: 竖屏计算器布局 (显示区 + 按键区)
- **主题**: 浅色/深色双主题支持
- **显示区**: 顶部显示当前输入和结果
- **按键区**: 4x5 网格布局

### 2.3 非功能需求

- **性能**: 启动时间 < 2 秒
- **包体积**: < 30MB
- **兼容性**: Android 7.0+

---

## 三、技术方案

### 3.1 架构设计

```
┌─────────────────┐
│   UI 层 (Flet)  │  ← 页面、组件、事件
├─────────────────┤
│  业务逻辑层      │  ← 计算引擎、表达式解析
├─────────────────┤
│   数据层        │  ← 历史记录存储 (本地)
└─────────────────┘
```

### 3.2 核心模块

| 模块 | 职责 |
|------|------|
| `main.py` | 应用入口、页面路由 |
| `calculator.py` | 计算逻辑、表达式处理 |
| `history.py` | 历史记录管理 |
| `theme.py` | 主题管理 |

### 3.3 打包方案

使用 Flet 官方命令行工具:

```bash
# 安装 flet
pip install flet

# 打包 APK (Linux 环境需配置 Android SDK)
flet build apk --release --project-name fletcalc
```

**环境要求**:
- Python 3.8+
- Flet 0.21.x
- Android SDK (如果本地打包)
